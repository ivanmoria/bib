Como usar

01
Gere um código de sessão Clique em "Gerar novo" acima. O código de 4 dígitos identifica a sessão.
02
Abra o Painel de Controle No seu dispositivo (professor). A tela de bloqueio pedirá o código da sessão.
03
Compartilhe o link da Tela Cada participante abre tela.html?id=XXXX no próprio dispositivo e clica em Conectar. O código já vem preenchido no link gerado acima.
04
Digite o código no Controle Insira o mesmo código na tela de bloqueio do Painel de Controle. A conexão com todos os participantes é estabelecida automaticamente.
05
Atribua os papéis e envie Use o slider ou as pílulas para definir o papel de cada participante. Clique em Enviar ou use o modo aleatório para distribuir automaticamente.
